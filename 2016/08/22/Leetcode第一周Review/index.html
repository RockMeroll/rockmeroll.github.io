292. Nim Game

一堆石子,两个人轮流取,一次可取1-3个,取完的人胜,题目很简单,但是思路一点都不简单,如果不可连续取(如[1,3,5])时该怎么解?。

当n∈[1,3]时，先手必胜。
当n = 4时，无论先手第一轮如何选取，下一轮都会转化为n∈[1,3]的情形，此时先手必负。
当n∈[5,7]时，先手必胜，先手分别通过取走[1,3]颗石头，可将状态转化为n == 4时的情形，此时后手必负。

258. Add Digits

将一个数n的每一位相加,直到结果为一位数为止,要求不使用循环(O(1))
数学题,将n除9取余就可以:
设一个四位数 4231
4231 = 4000 + 200 + 30 + 1
4000 % 9 = 4
200 % 9 = 2
30 % 9 = 3
4231 % 9 = 4000 % 9 + 200 % 9 + 30 % 9 + 1 % 9 = 4+2+3+1

382. Linked List Random Node

算法题,要求在链表中随机选出一个数据.
要使用C++的rand函数的话,要先知道链表的长度,效率很低.这里使用蓄水池算法,按顺序遍历链表,将第一个数据作为初始值,以 1/2 的概率决定要不要用第二个值替换结果,以 1/3 的概率决定要不要用第三个值替换结果,以此类推,这样每一个值都有相同的概率被选中.很精妙的算法!

238. Product of Array Except Self

Given an array of n integers where n > 1, nums, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i].
Solve it without division and in O(n).
For example, given [1,2,3,4], return [24,12,8,6].
