237. Delete Node in a Linked List

Write a function to delete a node (except the tail) in a singly linked list, given only access to that node.

Supposed the linked list is 1 -> 2 -> 3 -> 4 and you are given the third node with value 3, the linked list should become 1 -> 2 -> 4 after calling your function.

基础数据结构题，要求删除链表中的一个节点。

100. Same Tree

Given two binary trees, write a function to check if they are equal or not.

Two binary trees are considered equal if they are structurally identical and the nodes have the same value.

也是基础数据结构体，判断两个二叉树是否相等，后序遍历即可。

347. Top K Frequent Elements

Given a non-empty array of integers, return the k most frequent elements.

For example,
Given [1,1,1,2,2,3] and k = 2, return [1,2].

算法题，要求求出数组中出现次数最多的K个数。结果返回一个数组，很有价值的题目。

我的解法:
很蠢的解法，建一个MAP，数字n作为key，n出现的次数作为value，遍历数组得到MAP，在用sort函数进行排序，得出结果。O(nlogn)的解法。

优秀解法一: using heap
算是我的(愚蠢的)解法的优化版本。
priority_queue调用 STL里面的 make_heap(), pop_heap(), push_heap() 算法实现，也算是堆的另外一种形式。
STL map与Boost unordered_map的区别详见http://blog.csdn.net/orzlzro/article/details/7099231简而言之就是不需要排序的情况使用unordered_map更快。

优秀解法二: using selection algorithm

详细的选择算法:http://blog.csdn.net/lxiaoxiaot/article/details/6684727

优秀解法三: using bucket sort
经典排序算法桶排序 http://baike.baidu.com/view/1784217.htm

虽然我不是追求一题多解的那种大神，但是看了这些解法还是服气啊。。。

122. Best Time to Buy and Sell Stock II

Say you have an array for which the ith element is the price of a given stock on day i.

Design an algorithm to find the maximum profit. You may complete as many transactions as you like (ie, buy one and sell one share of the stock multiple times). However, you may not engage in multiple transactions at the same time (ie, you must sell the stock before you buy again).

很蠢的题目。。单调递增区间的累加而已。

383. Ransom Note

勒索信，题目描述很有意思，因为勒索信为了不暴露笔迹，会在杂志上剪下字来贴在纸上，就有了这道题。
其实原理很简单，就是两个字符串，验证字符串b中的字符是否能组成字符串a。
我用的老办法，使用两个map统计字符的个数，字符作为key，出现的次数作为value，然后要求mapNote中所有的value必须小于mapMagazine中的value。

357. Count Numbers with Unique Digits

Given a non-negative integer n, count all numbers with unique digits, x, where 0 ≤ x < 10n.

Example:
Given n = 2, return 91. (The answer should be the total numbers in the range of 0 ≤ x < 100, excluding [11,22,33,44,55,66,77,88,99])

算是数学题吧，要求出每一位都不同的数字个数。
高中的概率题，n位数的话，第一位有9个可能(0 不能作为第一位)，第二位有9种可能，第三位有8种。。。以此类推就可以。
另：十位数以上就不存在这样的数了

343. Integer Break

Given a positive integer n, break it into the sum of at least two positive integers and maximize the product of those integers. Return the maximum product you can get.

For example, given n = 2, return 1 (2 = 1 + 1); given n = 10, return 36 (10 = 3 + 3 + 4).
也是高中数学题，把一个数拆成多个数的和，要求他们的积最大
为了方便使用导数，我们先假设我们可以把n拆分成实数。那么设每一个数为x,则一共有n/x个数。
设它们的积为f(x),则f(x)=x(n/x)，再求导：
f′(x)=(n/x2) x(n/x) (1-lnx)
当x=e时取极大值。
而我们题目里规定x为整数，那么我们只需要取的x越靠近e越好。那么2

class Solution {
public:
    int integerBreak(int n) {
        if(n==2) return 1;
        if(n==3) return 2;
        
        if(n%3==0) return pow(3,n/3);
        else if(n%3==1) return 4*pow(3,(n-4)/3);
        else return 2*pow(3,n/3);
